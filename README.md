# CampusCompare

A modern college comparison platform built with Next.js, TypeScript, Tailwind CSS, Zustand, and dynamic API routing.

## Live Demo

https://campus-compare-mocha.vercel.app/

## Features

- Search colleges by name or location
- Filter colleges by location and rating
- Compare multiple colleges side-by-side
- Dynamic college detail pages
- Responsive design for mobile and desktop
- Loading skeleton states
- Clean reusable component architecture
- API-based data fetching

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Zustand
- Vercel
